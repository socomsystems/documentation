cyfrSpaces Documentation Overview¶

cyfrSpaces Server¶

You can download and install cyfrSpaces on your own Linux server, use the Web Installer to install it on shared Web hosting, try some prefab cloud or virtual machine images, or sign up for hosted cyfrSpaces services. See the Get Started page for more information.
